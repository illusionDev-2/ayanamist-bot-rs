# Ayanamistbot-rs

Ayanamistbot-tsのRustによる再実装

## Setup

### clone repo
```bash
git clone https://github.com/HaruSameee/ayanamist-bot-rs.git
cd ayanamist-bot-rs
cp config.toml.example config.toml
cargo run
```
